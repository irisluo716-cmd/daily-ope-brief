## 待办明细

### A. 内容质量（杠杆最大）
- [ ] **A1 事实接地 / 反幻觉**：成稿里每个数字/硬结论必须能回链到某条采集来源 URL，查不到就打回重写。对应 CLAUDE.md「推不出来宁可不写」。*（最高优先，关乎信任）*
- [ ] **A2 每条加一句"对隆鑫的含义"**：把通用新闻落到"对隆鑫的影响 / 动作建议 / 窗口"。摘要→情报的关键。
- [ ] **A3 推测标注分层**：官方一手 / 媒体聚合 / 用户原声 用不同标记，一眼区分能直接信 vs 只是风向。
- [ ] **A4 跨天去重 + 新增/延续**：加跨天 diff，只突出"今日新增"，延续中的折叠成一行。提纯、去重复刷屏。

### B. 数量 / 覆盖
- [ ] **B1 补欧洲源**：EU Safety Gate（召回）、Meteoalarm（预警）、Stage V（排放）。目标市场北美+欧洲，现严重偏美。
- [ ] **B2 补"中国供给侧" lane**：广交会新机型 / 委外商（大江·神驰·润通）动向 / 出口海关数据。需求侧+供给侧闭环，接 `资产库/B端对手库`。
- [ ] **B3 新增源类型（择优、只接稳定的）**：
  - 电商评论 VoC（Amazon/Home Depot/Lowe's）——最富用户原声，无干净 API，暂续走人工 Browser MCP；
  - 零售在售新 SKU——接 `资产库/对手SKU库`，自动发现对手新品；
  - 专利（USPTO/EPO）——对手 R&D 早信号，尤其割草机器人导航；
  - 财报 / 电话会（Generac/Husqvarna/Toro）——现仅抓到股价碎讯，没抓实质。

### C. 形式 / 体验
- [ ] **C1 Top 3 必看 + 其余折叠**：AI 每天先顶 3 条"今天只看这些"，不再每条等权。
- [ ] **C2 周 / 月滚动汇总**：周五"本周要点" + 月度"趋势"，喂现有「周报归档 → 季度小结 → BP」链路。
- [ ] **C3 给领导的可分享版**：按 LONCIN CI 出周报 PPT/Word（已有模板约定），领导读法不同于日报。

### D. 后续用途（最战略，系统终局）
- [ ] **D1 存档升级成"立项/五看证据库"**：按品牌/对手/关税码检索 + 时间线（底层 JSON 已支持）。立项时直接调"某对手半年所有动作 / 用户痛点演变"。
- [ ] **D2 竞品/关键词触发告警**：不只每天读，"一旦出现 Mammotion 新品 / 某关税码 / 某召回，立刻单独推我"。
- [ ] **D3 回写资产库**：发现对手新 SKU → 自动起草卡片进 `对手SKU库`；海关/市场数 → 追加进 `行业数据`。新闻系统与知识库打通。
- [ ] **D4 趋势 / 早警层**：召回频次、价格带下探速度、关税升级时间线——量化、可画图，喂 `事业部/行业走势判断` 的战略赌注检验。

## 两条原则（动手时守住）
1. **做减法优先**：Top3 + 去重 + 对隆鑫含义 这类"提纯"，比"加源"更值钱——先做减法再加量。
2. **只加稳定源**：加源只加有 API/feed 的；脆的（电商评论）继续走人工，别让爬虫拖垮维护。
